## 🚀 Описание проекта

**ГосБанк** — это симуляция интернет-банка, где пользователи могут:

- Подавать заявки на кредиты и овердрафты
- Оформлять дебетовые и кредитные карты
- Просматривать баланс и историю операций
- Входить через ЕСИА (Госуслуги) по упрощённому сценарию

Администраторы системы имеют доступ к панели управления для регистрации новых сотрудников.

> **Важно:** Проект является **учебным** и не предназначен для реального использования в финансовой сфере.

## ✨ Функциональные возможности

### 👤 Для пользователей
| Страница | Описание |
|----------|----------|
| **Главная** | Приветственный блок с кнопками входа и перехода в личный кабинет |
| **Карты** | Выбор и оформление дебетовой / кредитной карты |
| **Кредиты** | Заявка на кредит наличными или микрозайм |
| **Овердрафт** | Калькулятор овердрафта с примерным расчётом платежа |
| **Личный кабинет** | Просмотр баланса, истории операций, выход из системы |
| **Вход** | Авторизация через Госуслуги (упрощённая) или через логин/пароль для администратора |

### 👨‍💼 Для администраторов
| Функция | Описание |
|---------|----------|
| **Вход** | Отдельный вход через логин и пароль |
| **Панель администратора** | Регистрация новых администраторов через форму |
| **Валидация** | Проверка полей: email, возраст, пароль (латиница + цифры), согласие на обработку данных |

## 🏁 Инструкция по запуску

1. **Скачайте** все файлы проекта в одну папку.
2. **Откройте** `index.html` в браузере (двойным кликом или через `File → Open`).
3. **Авторизуйтесь:**
   - Как пользователь: нажмите «Войти через Госуслуги» → перейдёте в личный кабинет.
   - Как администратор: на странице входа переключитесь на «Вход для администратора» → введите любые логин/пароль → попадёте в панель администратора.
4. **Тестируйте** функционал: оформляйте карты, кредиты, овердрафт, регистрируйте новых админов.

> **Примечание:** Все данные хранятся в `localStorage`. При закрытии вкладки сессия сохраняется.

## 🎬 Сценарий работы пользователя

1. **Главная страница**  
   Пользователь видит заголовок: *«Много денег. Быстро.»* и предложение: *«Получите до 12 500 000 ₽ уже сегодня с овердрафтом»*.

2. **Выбор действия**  
   Кнопки: «Взять деньги сейчас» → перенаправляет на страницу кредитов.  
   «Рассчитать овердрафт» → открывает калькулятор.

3. **Авторизация через Госуслуги**  
   При попытке получить деньги система просит войти через ЕСИА.  
   Пользователь нажимает «Войти через Госуслуги» → попадает на форму входа.

4. **Вход**  
   Вводит логин (email) и пароль (упрощённая валидация).  
   После успешного входа переходит в личный кабинет с балансом и операциями.

5. **Администратор**  
   Администратор входит через отдельную форму, получает доступ к регистрации новых сотрудников.

## 🧪 Тестирование

### Позитивные тест-кейсы
- **TC1:** Успешная регистрация с минимальными значениями (email, возраст 19, пароль 8 символов).
- **TC2:** Успешная регистрация с максимальными значениями (возраст 100, пароль 256 символов).
- **TC3:** Выбор пола «Другой», город на латинице.

### Негативные тест-кейсы
- **TC4:** Пароль менее 8 символов → ошибка, кнопка неактивна.
- **TC5:** Пароли не совпадают → подсветка поля, ошибка.
- **TC6:** Не установлен флажок согласия на обработку данных → форма не отправляется.

> Полная матрица тестирования — в файле `Матрица.xlsx`.
